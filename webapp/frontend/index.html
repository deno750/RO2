🏃 Traveling Salesman Problem Optimization 🏃

This simple website allows you to find the shortest possible route that visits each city exactly once with different algorithms 🔥

Usage:
  1. Select the TSP instance or upload yours (it must be in the TSPLIB Format)
  2. Then Select the TSP method, time limit and ranom seed.
  3. Click on the Optimize button 💪 and wait ⏰ (time limit choosen)
⚠️ You may have to wait several minutes to get the response, don't close the web page. 😊
